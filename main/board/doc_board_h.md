# board.h

**Путь:** `main/board/board.h`

**Назначение:** Аппаратные определения (константы) платы HMI-дисплея установки обратного осмоса.

---

## Описание

Заголовочный файл `board.h` содержит все аппаратно-зависимые константы для платформы ESP32-P4 с 10.1-дюймовым дисплеем. Здесь определены: назначение GPIO-пинов, параметры MIPI DSI интерфейса, характеристики LCD-панели (контроллер JD9365), настройки сенсорного контроллера GSL3680 (I2C), параметры питания PHY и размеры буферов отрисовки LVGL.

Файл содержит условную компиляцию `#ifndef LVGL_LIVE_PREVIEW`: при сборке в режиме предпросмотра LVGL подключаются заглушки GPIO вместо реальных драйверов ESP-IDF.

---

## Определения (defines)

### Дисплей MIPI DSI (контроллер JD9365)

| Макрос | Значение | Описание |
|--------|----------|----------|
| `BOARD_LCD_H_RES` | 800 | Горизонтальное разрешение (нативное, до поворота) |
| `BOARD_LCD_V_RES` | 1280 | Вертикальное разрешение (нативное, до поворота) |
| `BOARD_LCD_MIPI_DSI_LANES` | 2 | Количество линий данных MIPI DSI |
| `BOARD_LCD_MIPI_LANE_BITRATE` | 1000 | Скорость одной линии MIPI DSI, Мбит/с |
| `BOARD_LCD_COLOR_SPACE` | `ESP_LCD_COLOR_SPACE_RGB` | Цветовое пространство |
| `BOARD_LCD_BPP` | 16 | Глубина цвета: RGB565 (16 бит на пиксель) |
| `BOARD_LCD_BIGENDIAN` | 0 | Порядок байт: little-endian |
| `BOARD_LCD_BACKLIGHT_GPIO` | `GPIO_NUM_23` | GPIO управления подсветкой (ШИМ через LEDC) |
| `BOARD_LCD_RST_GPIO` | `GPIO_NUM_27` | GPIO аппаратного сброса LCD-панели |

### Питание PHY MIPI DSI (внутренний LDO ESP32-P4)

| Макрос | Значение | Описание |
|--------|----------|----------|
| `BOARD_MIPI_DSI_PHY_LDO_CHAN` | 3 | Канал встроенного LDO-регулятора |
| `BOARD_MIPI_DSI_PHY_LDO_MV` | 2500 | Напряжение питания PHY: 2.5 В |

### Сенсорная панель (контроллер GSL3680, I2C)

| Макрос | Значение | Описание |
|--------|----------|----------|
| `BOARD_TOUCH_I2C_PORT` | `I2C_NUM_0` | Порт I2C (I2C0) |
| `BOARD_TOUCH_I2C_SDA` | `GPIO_NUM_7` | GPIO линии данных I2C SDA |
| `BOARD_TOUCH_I2C_SCL` | `GPIO_NUM_8` | GPIO линии тактирования I2C SCL |
| `BOARD_TOUCH_I2C_FREQ_HZ` | 400000 | Частота I2C: 400 кГц (Fast Mode) |
| `BOARD_TOUCH_RST_GPIO` | `GPIO_NUM_22` | GPIO аппаратного сброса тач-контроллера |
| `BOARD_TOUCH_INT_GPIO` | `GPIO_NUM_21` | GPIO прерывания от тач-контроллера |

### Ландшафтный режим (после программного поворота)

| Макрос | Значение | Описание |
|--------|----------|----------|
| `BOARD_DISP_H_RES` | 1280 | Горизонтальное разрешение в ландшафтном режиме |
| `BOARD_DISP_V_RES` | 800 | Вертикальное разрешение в ландшафтном режиме |

### Буфер отрисовки LVGL

| Макрос | Значение | Описание |
|--------|----------|----------|
| `BOARD_LCD_DRAW_BUFF_LINES` | 80 | Количество строк в буфере отрисовки |
| `BOARD_LCD_DRAW_BUFF_SIZE` | 800 * 80 = 64000 | Размер буфера в пикселях |

## Назначение GPIO

| GPIO | Функция | Направление |
|------|---------|-------------|
| GPIO_NUM_7 | I2C SDA (тач-контроллер) | Двунаправленный |
| GPIO_NUM_8 | I2C SCL (тач-контроллер) | Выход |
| GPIO_NUM_21 | Прерывание тач-контроллера | Вход |
| GPIO_NUM_22 | Сброс тач-контроллера | Выход |
| GPIO_NUM_23 | Подсветка LCD (ШИМ) | Выход |
| GPIO_NUM_27 | Сброс LCD-панели | Выход |

---

## Режим предпросмотра LVGL

При определении макроса `LVGL_LIVE_PREVIEW` вместо реальных драйверов GPIO и I2C используются числовые заглушки. Это позволяет собирать и предпросматривать UI на ПК без реального оборудования ESP32-P4.

## Связь с другими модулями

- **display_init.c** — использует макросы `BOARD_LCD_*` и `BOARD_MIPI_DSI_*` для настройки дисплея
- **touch_init.c** — использует макросы `BOARD_TOUCH_*` для настройки сенсорного контроллера
- **ui_main.c** — использует `BOARD_DISP_H_RES` / `BOARD_DISP_V_RES` для компоновки UI
