# Документация: ui_fonts

## Файл

- **Заголовок:** `main/fonts/ui_fonts.h`
- **Файлы шрифтов:** `main/fonts/Montserrat_*_4.c` (сгенерированные)

## Назначение

Декларации шрифтов для пользовательского интерфейса и удобные макросы-псевдонимы. Обеспечивает единообразное использование шрифтов во всех модулях UI через макросы `UI_FONT_XX`.

## Подробное описание

### Семейство шрифтов

Используется шрифт **Montserrat** -- геометрический гротеск с хорошей читаемостью на экранах. Все шрифты генерируются с глубиной антиалиасинга 4 бита (16 уровней серого), что обеспечивает плавные контуры на цветном дисплее.

### Именование

Формат имени: `Montserrat_<size>_<bpp>`

- `<size>` -- размер в пунктах (12, 14, 16, 18, 20, 22, 24, 28, 36)
- `<bpp>` -- бит на пиксель антиалиасинга (всегда 4)

### Набор символов

Все шрифты содержат:
- **Latin** (ASCII + расширенная латиница) -- для английского интерфейса
- **Cyrillic** -- для русского интерфейса
- **Цифры и спецсимволы** -- единицы измерения, знаки пунктуации

### Размеры и назначение

Размеры оптимизированы для дисплея 10.1" (1280x800, ~150 DPI):

| Макрос | Размер | Назначение |
|---|---|---|
| `UI_FONT_12` | 12pt | Мелкие подписи, единицы измерения на мнемосхеме |
| `UI_FONT_14` | 14pt | **Шрифт по умолчанию** для темы LVGL. Мелкий текст |
| `UI_FONT_16` | 16pt | Вторичный текст, подписи в таблицах |
| `UI_FONT_18` | 18pt | Табличные данные, списки параметров |
| `UI_FONT_20` | 20pt | Основной текст, метки параметров |
| `UI_FONT_22` | 22pt | Названия параметров, текст в диалогах подтверждения |
| `UI_FONT_24` | 24pt | Заголовки экранов |
| `UI_FONT_28` | 28pt | Крупные числовые значения |
| `UI_FONT_36` | 36pt | Ключевые показатели на мнемосхеме |

## Макросы

### Декларации шрифтов (LV_FONT_DECLARE)

```c
LV_FONT_DECLARE(Montserrat_12_4);
LV_FONT_DECLARE(Montserrat_14_4);
LV_FONT_DECLARE(Montserrat_16_4);
LV_FONT_DECLARE(Montserrat_18_4);
LV_FONT_DECLARE(Montserrat_20_4);
LV_FONT_DECLARE(Montserrat_22_4);
LV_FONT_DECLARE(Montserrat_24_4);
LV_FONT_DECLARE(Montserrat_28_4);
LV_FONT_DECLARE(Montserrat_36_4);
```

Макрос `LV_FONT_DECLARE` из LVGL создаёт объявление `extern const lv_font_t <name>`, позволяя ссылаться на шрифт до его определения в .c файле.

### Псевдонимы (UI_FONT_XX)

```c
#define UI_FONT_12  &Montserrat_12_4
#define UI_FONT_14  &Montserrat_14_4
// ... и т.д.
#define UI_FONT_36  &Montserrat_36_4
```

Макросы возвращают указатель `const lv_font_t*`, готовый для использования в стилях LVGL:
```c
lv_obj_set_style_text_font(label, UI_FONT_22, 0);
```

## Интеграция с LVGL

- **LV_FONT_DECLARE:** Стандартный макрос LVGL v9 для объявления внешнего шрифта
- **const lv_font_t:** Каждый шрифт -- статическая константная структура, определённая в .c файле
- **Антиалиасинг:** 4-битный режим (16 уровней) -- оптимальный баланс между качеством и расходом памяти
- **Генерация:** Файлы шрифтов `.c` генерируются инструментом `lv_font_conv` или встроенным генератором LVGL
