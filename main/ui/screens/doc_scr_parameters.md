# Экран "Параметры" (scr_parameters)

## Файлы
- `main/ui/screens/scr_parameters.h` -- заголовок (API)
- `main/ui/screens/scr_parameters.c` -- реализация

## Назначение
Таблица всех измеренных и расчётных величин установки обратного осмоса.
Позволяет оператору видеть точные значения всех параметров с указанием статуса каждого канала.

## Иерархия виджетов

```
cont (1280x700, FLEX_COLUMN, прокручиваемый)
  +-- hdr_row (FLEX_ROW) -- заголовки столбцов
  +-- section_header "Давление"
  +-- row P1 (FLEX_ROW): lbl_name | lbl_p_val[0] | (пусто) | lbl_p_status[0]
  +-- row P2..P4 (аналогично)
  +-- section_header "Температура"

  +-- row calc[0..5]: lbl_name | lbl_calc_val[i] | (пусто) | (пусто)
```

## Функции

| Функция | Описание |
|---|---|
| `scr_parameters_create(parent)` | Создаёт таблицу с секциями, возвращает контейнер |
| `scr_parameters_update(cont, data)` | Обновляет значения всех строк |
| `make_section_header(...)` | Заголовок секции (акцентный цвет) |
| `make_row(...)` | Одна строка: имя, значение, доп.столбец, статус |

## Источники данных (plant_data_t)

| Поле | Столбец Value | Столбец Extra | Статус |
|---|---|---|---|
| `pressure[i].value/.fault` | bar (3 зн.) | -- | OK / Обрыв датчика |
| `temperature.value/.fault` | C (1 зн.) | -- | OK / Обрыв датчика |
| `flow[i].flow` | m3/h (3 зн.) | `flow[i].volume` m3 | OK / Неисправность |
| `conductivity[i].conductivity` | uS/cm (1 зн.) | `.temperature` C | OK / Неисправность |
| `telemetry.filter_dp` | bar (3 зн.) | -- | -- |
| `telemetry.stage1_feed` | m3/h (2 зн.) | -- | -- |
| `telemetry.recovery2` | % (1 зн.) | -- | -- |
| `telemetry.recovery_sys` | % (1 зн.) | -- | -- |
| `telemetry.sel1` | % (1 зн.) | -- | -- |
| `telemetry.sel2` | % (1 зн.) | -- | -- |

## Взаимодействие пользователя
Экран только для чтения. Нет кнопок и интерактивных элементов.
Поддерживается вертикальная прокрутка (touch scroll).

## Механизм обновления
- Периодический (таймер ~500 мс в ui_main.c)
- `scr_parameters_update()` вызывается из UI-таймера
- Обновляет текст и цвет каждой строки
