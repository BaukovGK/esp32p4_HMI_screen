# Документация: ui_theme

## Файлы

- **Заголовок:** `main/ui/ui_theme.h`
- **Реализация:** `main/ui/ui_theme.c`

## Назначение

Модуль темы оформления UI. Определяет единую цветовую палитру "Industrial Dark" для всех экранов HMI-дисплея и предоставляет функции маппинга состояний установки и категорий аварий на цвета LVGL.

## Подробное описание

### Цветовая схема "Industrial Dark"

Тёмная цветовая схема, оптимизированная для промышленного HMI-дисплея:
- Хорошая читаемость при различном освещении
- Контрастные цвета состояний для быстрого визуального распознавания
- Светлый текст на тёмном фоне снижает нагрузку на глаза оператора

## Функции

### `void ui_theme_init(void)`

Инициализация темы LVGL. Выполняет:
1. Получение дисплея по умолчанию через `lv_display_get_default()`
2. Создание стандартной тёмной темы LVGL через `lv_theme_default_init()` с параметрами:
   - Акцентный цвет: `COLOR_ACCENT` (голубой)
   - Вторичный цвет: `COLOR_BTN_PRIMARY` (синий)
   - Тёмный режим: `true`
   - Шрифт по умолчанию: `UI_FONT_14` (Montserrat 14pt)
3. Применение темы к дисплею через `lv_display_set_theme()`
4. Установка тёмного фона (`COLOR_BG_DARK`) на активном экране

### `lv_color_t ui_state_color(int state)`

Маппинг состояния установки на цвет индикации.

**Параметр:** `state` -- значение `plant_state_t` (IDLE=0, AUTO=1, WASHING=2, MANUAL=3, FAULT=4)

**Возвращает:** цвет LVGL:
- `PLANT_STATE_IDLE` -> `COLOR_STATE_IDLE` (серый)
- `PLANT_STATE_AUTO` -> `COLOR_STATE_AUTO` (зелёный)
- `PLANT_STATE_WASHING` -> `COLOR_STATE_WASHING` (синий)
- `PLANT_STATE_MANUAL` -> `COLOR_STATE_MANUAL` (жёлтый)
- `PLANT_STATE_FAULT` -> `COLOR_STATE_FAULT` (красный)
- По умолчанию -> `COLOR_STATE_IDLE` (серый)

### `lv_color_t ui_alarm_cat_color(int cat)`

Маппинг категории аварии на цвет индикации.

**Параметр:** `cat` -- значение `alarm_category_t` (INFO=0, WARNING=1, ALARM=2, CRITICAL=3)

**Возвращает:** цвет LVGL:
- `ALARM_CAT_CRITICAL` -> `COLOR_ALARM_CRITICAL` (красный)
- `ALARM_CAT_ALARM` -> `COLOR_ALARM_ALARM` (оранжевый)
- `ALARM_CAT_WARNING` -> `COLOR_ALARM_WARNING` (жёлтый)
- По умолчанию -> `COLOR_ALARM_INFO` (серый)

| Модуль | Назначение |

## Интеграция с LVGL

- **Тема по умолчанию:** Используется `lv_theme_default_init()` из LVGL v9 -- стандартная тема с пользовательскими параметрами
- **Тёмный режим:** Параметр `dark = true` при инициализации темы
- **Цвета как макросы:** Все цвета определены как `lv_color_hex()` макросы для использования в стилях через `lv_obj_set_style_*_color()`
- **Без глобального состояния:** Функции `ui_state_color()` и `ui_alarm_cat_color()` чисто функциональные (без побочных эффектов), могут вызываться из любого контекста

## Использование в других модулях

Цветовые макросы из `ui_theme.h` используются во всех компонентах UI:
- `ui_common.c` -- панель аварий, навигация
- `ui_events.c` -- диалог подтверждения
- `scr_mnemonic.c` -- мнемосхема (индикаторы насосов, состояния)
- `scr_parameters.c` -- карточки параметров
- `scr_alarms.c` -- таблица аварий
- `scr_settings.c` -- формы настроек
- `scr_diagnostics.c` -- диагностическая информация
