# ТЕХНИЧЕСКОЕ ЗАДАНИЕ
# Система управления опытно-промышленной установкой обратного осмоса

**Версия:** 2.0  
**Дата:** 2026-02-04  
**Статус:** Финализировано

---

## 1. НАЗНАЧЕНИЕ СИСТЕМЫ

Автономный контроллер двухступенчатой опытно-промышленной установки обратного осмоса для:
- Непрерывной работы установки в автоматическом режиме
- Записи телеметрии для последующей статистической обработки
- Проведения химических промывок мембран
- Пусконаладочных работ в ручном режиме

### 2.2 Насосное оборудование

| № | Насос | Назначение | Управление |
|---|-------|------------|------------|
| 1 | Насос преднагнетания | Забор воды из исходной ёмкости, подача на фильтр, циркуляция при химмойке | RO1 (УПП) |
| 2 | Насос 1-й ступени (ВД) | Подача воды на мембраны 1-й ступени | RO2 (УПП) |
| 3 | Насос 2-й ступени (ВД) | Подача воды на мембраны 2-й ступени | RO3 (УПП) |
| 4 | Насос-дозатор | Дозирование реагентов | RO5 (сигнальный) |

Насосы 1-3 оснащены устройствами плавного пуска (УПП), время разгона 10-15 секунд.

### 2.3 Ёмкости

| Ёмкость | Датчики уровня | Назначение |
|---------|----------------|------------|
| Исходная | Низкий (DI1, NC) | Источник воды для установки |
| Промежуточная | Низкий (DI2, NC), Высокий (DI3, NO) | Буфер между ступенями |
| Пермеатная | Высокий (DI4, NO) | Накопление очищенной воды |

### 2.4 Потоки концентрата

| Источник | Направление | Управление |
|----------|-------------|------------|
| Концентрат 1-й ступени | Рецикл перед насосом 1-й ст. (Q2) | Ручной клапан |
| | Дренаж | Ручной клапан |
| Концентрат 2-й ступени | Рецикл на 2-ю ступень | Ручной клапан |
| | Рецикл на 1-ю ступень | Ручной клапан |

**Примечание:** Концентрат 2-й ступени не сбрасывается в дренаж — только рециркуляция.

### 2.5 Дополнительное оборудование

- **Нагреватель** (RO4) — для подогрева промывочного раствора до 35-40°C
- **Клапаны** — ручные (переключаются оператором)
- **Фильтр механической очистки** — между насосом преднагнетания и насосом 1-й ступени

## 4. РАСПРЕДЕЛЕНИЕ ВХОДОВ/ВЫХОДОВ

### 4.1 Дискретные входы (DI)

| Вход | Назначение | Тип контакта | Логика |
|------|------------|--------------|--------|
| DI1 | Уровень исходной — низкий | NC | 0 = пусто, 1 = есть вода |
| DI2 | Уровень промежуточной — низкий | NC | 0 = пусто, 1 = есть вода |
| DI3 | Уровень промежуточной — высокий | NO | 0 = не полная, 1 = полная |
| DI4 | Уровень пермеатной — высокий | NO | 0 = не полная, 1 = полная |
| DI5 | Аварийный стоп (кнопка) | NC | 0 = авария, 1 = норма |
| DI6 | Подтверждение насоса преднагнетания | NO | 1 = контактор включён |
| DI7 | Подтверждение насоса 1-й ступени | NO | 1 = контактор включён |
| DI8 | Подтверждение насоса 2-й ступени | NO | 1 = контактор включён |

### 4.2 Релейные выходы (DO)

| Выход | Назначение | Нагрузка |
|-------|------------|----------|
| RO1 | Насос преднагнетания | УПП |

| RO6-RO8 | Резерв | — |

| Канал | Назначение | Диапазон | Единицы |

| AI3 | Давление после насоса 1-й ступени (P3) | 4-20mA → 0-40 | bar |
| AI4 | Давление после насоса 2-й ступени (P4) | 4-20mA → 0-10 | bar |

## 5. MODBUS-УСТРОЙСТВА

### 5.1 Параметры шины RS-485

| Параметр | Значение |
|----------|----------|
| Скорость | 115200 бод |
| Data bits | 8 |
| Parity | None |
| Stop bits | 1 |
| Протокол | Modbus RTU |

### 5.2 Карта устройств

| Адрес | Устройство | Модель | Период опроса |
|-------|------------|--------|---------------|
| 1 | Модуль аналоговых входов | Waveshare Modbus RTU Analog Input 8CH | 100 мс |
| 2 | Расходомер (4 канала) | УРЖ2КМ-3.3 | 1000 мс (расход) / 2000 мс (объём) |
| 10 | Кондуктометр (2 канала) | СЛ21-201-100Т | 3000 мс |
| 11 | Кондуктометр (1 канал) | СЛ21-101-100Т | 3000 мс |

### 5.3 Расходомер УРЖ2КМ (адрес 2)

**Формат float:** нестандартный (байты переставлены)
```
Стандартный IEEE754: SEEEEEEE EAAAAAAA BBBBBBBB CCCCCCCC
УРЖ2КМ:              BBBBBBBB CCCCCCCC SEEEEEEE EAAAAAAA
```

**Назначение каналов:**

| Канал | Расположение | Назначение |
|-------|--------------|------------|
| Q1 | После насоса преднагнетания | Входной расход |
| Q2 | Рецикл концентрата 1-й ступени | Возврат перед насосом 1-й ст. |
| Q3 | Пермеат 2-й ступени | Выход чистой воды |
| Q4 | Концентрат 2-й ступени | Рециркуляция |

**Регистры:** 0x0000-0x0007 (расход), 0x0036-0x0045 (объём), 0x0072-0x0073 (статус)

### 5.4 Кондуктометры СЛ21 (адреса 10, 11)

**Распределение каналов:**

| Блок | Адрес | Канал 1 | Канал 2 |
|------|-------|---------|---------|
| СЛ21-201-100Т | 10 | σ1 — После насоса преднагн. | σ2 — Пермеат 1-й ступени |
| СЛ21-101-100Т | 11 | σ3 — Пермеат 2-й ступени | — |

**Масштабирование:** УЭП ÷100 → мкСм/см, температура ÷10 → °C

### 5.5 Waveshare Analog Input 8CH (адрес 1)

**Функция Modbus:** 0x04 (Read Input Registers)  
**Регистры:** 0x0000-0x0007

## 6. РЕЖИМЫ РАБОТЫ

### 6.1 Режим AUTO (Автоматическая фильтрация)

**Логика управления по ёмкостям:**

| Событие | Реакция |
|---------|---------|
| Исходная пустая (DI1=0) | Останов преднагн. + 1-й ст. 2-я продолжает |
| Промежуточная полная (DI3=1) | Останов преднагн. + 1-й ст. 2-я продолжает |
| Промежуточная пустая (DI2=0) | Останов 2-й ступени |
| Пермеатная полная (DI4=1) | Останов 2-й ступени |

**Последовательность пуска:** Преднагн. → (подтв. + разгон + давление) → 1-я ст. → (заполнение промежут.) → 2-я ст. → Дозатор (если timer)

### 6.2 Режим WASHING (Химическая мойка)

| Фаза | Действия | Длительность |
|------|----------|--------------|
| Нагрев | Преднагн. ВКЛ, нагреватель ВКЛ (до 35°C) | 30 мин |
| Подача | Ручное переключение клапанов | 20 мин |
| Слив | Ручное открытие дренажа | — |

**Насос-дозатор НЕ работает в режиме WASHING.**

### 6.3 Режим MANUAL (Ручное управление)

Независимое управление всеми агрегатами с минимальными блокировками.

### 6.4 Насос-дозатор

| Режим | Поведение |
|-------|-----------|
| manual | По команде оператора |
| timer | N минут работы за M минут цикла (только в AUTO) |

## 7. БЛОКИРОВКИ И БЕЗОПАСНОСТЬ

| Блокировка          | Условие           | Реакция                     | Категория |
|---------------------|-------------------|-----------------------------|-----------|
| Аварийный стоп      | DI5=0             | Останов ВСЕХ                | CRITICAL  |
| Сухой ход исходной  | DI1=0             | Останов преднагн. + 1-я ст. | WARNING   |
| Сухой ход промежут. | DI2=0             | Останов 2-й ст.             | WARNING   |
| Превышение P1       | P1 > 5.5 bar      | Останов преднагнетания      | ALARM     |
| Превышение P3       | P3 > 35 bar       | Останов 1-й ступени         | ALARM     |
| Превышение P4       | P4 > 8 bar        | Останов 2-й ступени         | ALARM     |
| Засорение фильтра   | P1 - P2 > 1.0 bar | Предупреждение              | WARNING   |
| Нет подтверждения   | Таймаут DI6/7/8   | Авария                      | ALARM     |
| Перегрев            | T > 45°C          | Откл. нагревателя           | WARNING   |

## 9. РАСЧЁТНЫЕ ПАРАМЕТРЫ

| Параметр                   | Формула               |
|----------------------------|-----------------------|
| Расход на мембрану 1-й ст. | Q1 + Q2               |
| Recovery 2-й ступени       | Q3 / (Q3 + Q4) × 100% |
| Общий выход системы        | Q3 / Q1 × 100%        |
| Перепад на фильтре         | P1 - P2               |
| Селективность 1-й ст.      | (σ1 - σ2) / σ1 × 100% |
| Селективность 2-й ст.      | (σ2 - σ3) / σ2 × 100% |
